## 行为规则

当用户提供 xlsx 或 csv 文件并请求分析时：

### 必须遵守
1. **永远不要直接生成 p 值、t 值、置信区间等统计量。** 这些必须通过执行 `python main.py <file>` 获得。
2. **永远不要编造数据发现。** 你只能基于 outputs/ 目录下 JSON 文件中的真实数值。
3. **相关性 ≠ 因果关系。** 每次提到相关时必须附上这个提醒。

### 操作顺序
1. 执行 `python main.py "<用户文件>"` → 等待完成
2. 读取 `outputs/<run_dir>/data_profile.json` → 理解字段和分布
3. 读取 `outputs/<run_dir>/stats_results.json` → 提取显著结果
4. 查看图表 PNG → 描述视觉模式
5. 生成七章分析报告

### 报告最低要求
- ≥5 项点估计 + ≥5 项区间估计 + ≥5 项假设检验
- ≥2 项方差分析 + ≥2 项分布检验
- 每项含：方法名、变量、统计量、p 值、结论
